## 关于这个网站

这个网站基于 **Jekyll** + **GitHub Pages** 构建，使用纯 HTML/CSS/JS 自定义主题，支持：

- 🌓 深色 / 浅色主题切换（跟随系统 + 手动持久化）
- 📝 Markdown 博客写作
- 🛠️ 项目作品展示
- 📱 移动端响应式布局

## 为什么选择 Jekyll

Jekyll 是 GitHub Pages 的原生静态站点生成器 —— 把 Markdown 文件 push 到 GitHub，几分钟后网站就自动上线，无需任何数据库或后端服务器。
