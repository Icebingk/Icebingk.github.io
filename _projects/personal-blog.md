## 项目简介

基于 **Jekyll** 静态站点生成器的个人技术博客，部署于 GitHub Pages。支持浅色/深色双主题自动切换、响应式布局，展示个人项目作品与技术文章。

## 技术亮点

- **双主题系统**：CSS 变量 + JavaScript 实现系统级亮暗模式跟随，支持手动切换
- **Jekyll 集合机制**：利用 `collections` 管理项目展示，YAML Front Matter 驱动内容
- **响应式设计**：移动端友好的汉堡菜单导航，自适应布局
- **GitHub Pages 原生部署**：零成本托管，Markdown 撰写，Git 版本管理

## 技术栈

`Jekyll` `HTML/CSS` `JavaScript` `GitHub Pages` `Markdown` `Liquid`
